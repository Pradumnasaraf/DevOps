- `whereis` - Найти путь к исполняемому файлу.

### Операции со списком
- `ls` - Показывает список.
    - `-a` - Скрытые файлы.
    - `-l` - Разрешения.
    - `-R` - Показать поддиректории.

### Операции смены директории
- `cd <folder-name>` - Сменить директорию.
- `cd ..` -   Вернуться на одну директорию назад.
- `cd` -    Перейти в домашнюю директорию.
- `cd ../<foldername>` - Открыть папку в предыдущей директории.
- `cd <path>` - Открыть директорию по пути.

### Операции с файлами/папками
- `mkdir <new-dir-name>` - Создать новую папку.
- `mkdir -p test/test1/test2` - Создать директорию между двумя директориями.
- `touch <new-file-name>` - создать пустой файл.
- `pwd` - Текущая рабочая директория.
- `cat <filename>` - Отобразить содержимое файла.
- `cat > <new-file-name>` - Создать файл.
- `dd if=/dev/zero of=bos_dosya bs=4G count=1`- создать пустой файл с нулями
- `cat >> <filename>` - Добавить в файл
- `cat  <filename1> >> <filename2>` - Добавить содержимое файла filename1 в конец файла `<filename2>`.
- `cat <filename> <filename2> ` - Отобразить 2 файла одновременно.
- `cat <filename> <filename2> > <newfile-name>` - Объединить содержимое обоих файлов в один.
- `cat <file-name> | tr > <new-file-name>` - Перевести файл.
- `cut -c  1-2 <filename>` - обрезать файл по столбцам
- `echo "Hello" >> <file-name>`
- `man <commad name>` - Узнать об использовании команды и опциях.
- `man <commad name>` - узнать о команде.

### Операции с файлами/папками
- `cp <file-name> <new-fie-name>` - Сделать копию файла в текущем местоположении.
- `mv <file-name> <dir-path>` - Переместить файл из одной директории в другую.
- `mv <file-name> <new-fie-name>` - Переименовать файл.
- `mv -R <dir-name> <dir-path>` - Переместить директорию
- `rm <file-name>` - Удалить файл навсегда.
- `rm -R <file-name>` - Удалить папку с включенной директорией.
- `head <file-name>` - Покажет первые 10 строк файла.
- `tail <file-name>` - Покажет последние 10 строк файла.
    -`-n 2` - покажет последние 2 строки.
- `diff <file-1> <file-2>` - Показать разницу между двумя файлами.
- `locate <file>` - Чтобы найти файл.  
- `find <file/folder-name>` - Найти файл/папку.
- `find <dir-name>` - Найти файлы внутри директории
- `find .-type d` - Показать только директории.
    - `.-type f` - показать только файлы.
    - `.-type f -name "*.txt"` - Показать только файлы с этим конкретным именем.
    - `.-type f -iname "*.txt"` - Показать только файлы с этим конкретным именем - не чувствительно к регистру (i)
    - `.-type f -mmin -20` - Показать файлы, которые были изменены менее 20 минут назад.
    - `.-type f -mmin +20` - показать файлы, которые были изменены более 20 минут назад.
    - `.-type f -maxdepth 2` - Покажет только на 1 папку вглубь.
    - `.-size +1k` - покажет только файлы/папки размером 1kb

### Системные команды
- `ps aux` - процессы, которые запущены
- `systemctl [option] [service]` - взаимодействовать с процессом
    - Мы можем делать 4 `опции` с `systemctl`
        - start
        - stop
        - enable
        - disable
    - Пример, `systemctl start apache2`     
- `df` - Проверить емкость и детали хранилища.
    - `m` - В мегабайтах или 
    - `hg` - В гигабайтах.
- `du` - Использование диска
    - `-h` (человекочитаемый)
- `echo` - Получить вывод строки
- `echo $PATH` - Проверить переменную пути
- `sudo` - Админ команда
- `sudo chown root text.txt` - изменить владельца
- `!<command-name>` - Запустить предыдущую команду
- `git add .; git commit -m "message"` - Запустить несколько команд одновременно
- `sort <file-name>"` - отсортировать файл
- `job` - показать задачи
- `wget <url>` - скачать файл с URL
- `top` - какие процессы запущены
- `kill <process-id>` -остановить этот процесс
- `Uname` - показать информацию о системе
- `zip <file-1> <file-2>` - Заархивировать два или более файлов
- `Unzip <file-name>` - Разархивировать файлы
- `useradd <name>` - добавить пользователя
- `passwd <name>` - установить пароль для пользователя
- `uname -<flag>` -o -m -r
- `lscpu` - получить детали CPU
- `free` - свободная память 
- `vmstat` - виртуальная память
