- [WasmFiddle](https://wasdk.github.io/WasmFiddle/) - Онлайн площадка для WebAssembly.
- [WebAssembly Explorer](https://mbebenita.github.io/WasmExplorer/) - Онлайн инструмент для изучения WebAssembly.
- [WABT Online](https://webassembly.github.io/wabt/demo/wat2wasm/) - Онлайн инструмент для работы с WebAssembly.
- [WasmCloud Playground](https://playground.wasmcloud.dev/) - Онлайн инструмент для создания и запуска WebAssembly модулей.
