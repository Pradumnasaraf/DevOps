## YAML - YAML Ain't Markup Langugae

- Data format used to exchnage data.
- Similar to XML & JSON.
- We can't add commands.
- Simple and easy to read.
- Strict syntax - (Indentation)
- More powerful to represent complex data.
- Parsing is easy
